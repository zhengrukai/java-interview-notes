*
    ### `#{}`：预编译占位符（安全）

MyBatis 会把#{}替换成 SQL 中的?占位符，然后通过 JDBC 的PreparedStatement预编译 SQL，再通过setXxx()方法给?赋值。

相当于考试时的填空题：先印好 "SELECT \* FROM user WHERE id = ?"，再单独填答案，格式和安全都有保障。

*
    ### `${}`：字符串拼接（直接手写）（有注入风险）

MyBatis 会直接把\${}里的参数值原样拼接到 SQL 语句中，没有预编译过程。

相当于考试时直接把答案写进题目里：如果参数是 1，就变成 "SELECT \* FROM user WHERE id = 1"；如果参数是恶意值，也会直接拼进去。
